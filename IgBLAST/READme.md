# IgBLAST database files
IgBLAST database files contain combined IMGT Ig/TCR constant, variable, diversity, and joining germline sequences from mouse.
This IgBLAST database was used for ferret Ig/TCR constant region consensus sequence generation described by Adams et al. (X year).
IgBLAST database was built using [NCBI IgBLAST](https://ncbi.github.io/igblast/cook/How-to-set-up.html).
