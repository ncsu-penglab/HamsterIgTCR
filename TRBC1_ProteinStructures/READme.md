# TRBC1_ProteinStructures

This folder contains the highest-confidence **AlphaFold3-predicted hamster TRBC1 protein structures** for both the hamster **reference** sequence (genomic sequence matching the hamster reference genome) and an **alternate** sequence containing the SNV that results in a non-synonymous amino acid change.

## Files included:
- `fold_trbc1_213a_ref_hamster_model_0.cif`:  
  AlphaFold3-predicted structure of **hamster TRBC1 reference** (with nucleotide 213A).
  
- `fold_trbc1_213g_alt_hamster_model_0.cif`:  
  AlphaFold3-predicted structure of **hamster TRBC1 alternate variant** containing the SNV **213A>G**, which introduces a non-synonymous amino acid change **S71G**.
  
- `7amp.pdb`:  
  **Human TRBC1 crystal structure** (PDB ID: 7AMP), obtained from the Protein Data Bank. This structure contains only the extracellular domain.
