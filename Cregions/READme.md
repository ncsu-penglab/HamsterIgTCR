# Cregions

The **Hamster_IgTCR_Cregions.fasta** file contains constant region consensus sequences that were generated by the pipeline described by Adams et al. (X year).

IMGT independently analyzed the same long-read transcriptome datasets and have publicly available germline and artificially spliced
